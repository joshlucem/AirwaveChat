**AirwaveChat** is a radio chat plugin for Paper/Spigot/Folia 1.21+ servers, developed by **@joshlucem**. It allows players to communicate on private frequencies, simulating real radios in Minecraft.

---

## Download
You can always download the latest version of AirwaveChat from the [Releases page](https://github.com/joshlucem/AirwaveChat/releases/latest).

## Features
- Private chat by configurable frequencies
- Commands to connect, disconnect, and list frequencies
- Customizable messages
- Easy to install and configure

## Commands
| Command                   | Description                                      |
|---------------------------|--------------------------------------------------|
| `/connect <frequency>`    | Connect to a radio frequency.                    |
| `/disconnect`             | Disconnect from your current frequency.          |
| `/frequencies list`       | List available frequencies.                      |
| `/frequencies current`    | Show your current frequency.                     |
| `/frequencies info <frequency>` | Show info for a frequency.               |
| `/frequencies top`        | Show top frequencies by users.                   |
| `/airwavechat reload`     | Reload config and messages. (Admin)              |
| `/airwavechat about`      | Show plugin information. (Admin)                 |
| `/airwavechat help`       | Show command help.                               |

## Requirements
- Paper/Spigot/Folia server 1.21+
- Java 21+
